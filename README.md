# lingostickers-mobile

Learn a language through your own life: point your camera at a real object,
get a sticker with the word, how to say it, and a sentence about the scene —
building a personal, story-driven vocabulary collection over time.

See **[ROADMAP.md](./ROADMAP.md)** for the product vision, the phased plan
for what's being built next, and the background-removal scaling research.
